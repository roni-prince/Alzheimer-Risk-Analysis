# Alzheimer's Disease Analysis Project

This project analyzes lifestyle and clinical factors related to Alzheimer's Disease.

## Project Structure
- `main.py`: The entry point of the application.
- `src/`: Core logic modules (loading, cleaning, analysis).
- `data/`: Contains the dataset.
- `tests/`: Automated tests for data integrity.
